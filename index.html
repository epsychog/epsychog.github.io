WHY have a Plotting Predictors Application?

  • Easily visualise predictors from the Wage data
  • Understanding how the data looks & how it interacts at a glance
  • Quick access to a selection of the most popular plotting methods
  • Switch easily between plot types and compare results

WHAT is the Plotting Predictors Application?

  • A simple application that provides a number of the most popular predictor plots
  • Works on the Wage dataset
  • Offers a selection of 5 different plot types
  • The plot types available are:
    • Feature Plot (Wage): Feature plot using age, education and jobcode against wage
    • Jobclass (Age vs Wage): Plot of age vs wage coloured by jobclass.
    • Smoothers (Age vs Wage): Plot of age vs wage coloured by education with linear smoother for each education class
    • Boxplot (Wage vs Age): Boxplot of age vs wage showing with different colours the various wage ranges
    • Density Plot (Wage): Density plot of wage vs education

HOW does the Plotting Predictors Applications Work?

  • The user selects the plot type from a drop-down list
  • The code for the plot is generated
  • The plot appears on the right hand side of the screen
